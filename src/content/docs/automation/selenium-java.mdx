# Selenium + Java

## Objectives

- Set up a basic Selenium + Java project.
- Automate a simple web flow using good structure.

## Recommended stack

- Language: **Java**
- Build tool: **Maven** or **Gradle**
- Test runner: **JUnit** or **TestNG**

## Minimal flow

1. Open the application URL.
2. Perform a login.
3. Assert a key element on the landing page.

## Good practices

- Use the **Page Object Model** to separate locators from tests.
- Keep waits explicit and controlled.
- Make locators resilient (by data attributes where possible).
