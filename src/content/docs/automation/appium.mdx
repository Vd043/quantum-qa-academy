# Appium Basics

## Objectives

- Understand how Appium fits into mobile testing.
- Run a simple test on Android or iOS.

## Key concepts

- Appium server
- Desired capabilities
- Native vs. hybrid vs. web apps

## Basic workflow

1. Start Appium server.
2. Connect to a real device or emulator/simulator.
3. Launch app under test.
4. Interact with elements and assert outcomes.

## Tips

- Stabilize flakiness with proper waits and IDs.
- Keep Appium dependencies and device farm versions aligned.
