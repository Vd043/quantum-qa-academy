# QA SDLC Overview

The **QA Software Development Life Cycle (QA SDLC)** is the structured process QA follows alongside delivery teams.

## Phases

1. **Requirement analysis**
2. **Test planning**
3. **Test design**
4. **Environment & data preparation**
5. **Test execution**
6. **Reporting & closure**

## Deliverables per phase

- Requirement analysis → risk list, clarification questions.
- Planning → test strategy, test plan.
- Design → test cases, checklists, test data.
- Execution → execution logs, defect reports.
- Closure → test summary report, lessons learned.
